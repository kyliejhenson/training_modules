## Content

One of MCP's key strengths is being transport agnostic — the client and server can communicate over different protocols depending on your setup. The most common setup runs both the MCP client and server on the same machine, communicating through standard input/output. But you can also connect them over:

- HTTP
- WebSockets
- Various other network protocols
