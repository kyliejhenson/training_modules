## Content

The example project is a CLI-based chatbot that implements both an MCP client and server for educational purposes. It includes a fake in-memory document system with two tools: reading and updating document contents.

**Setup steps:**

1. Download and extract the starter project files
2. Configure `.env` with your Anthropic API key
3. Install dependencies (see project README)
4. Run with `uv run main.py` or `python main.py`

The UV package manager is used for dependency management. Install it from https://docs.astral.sh/uv/.
