## Content

MCP shifts the burden of tool integration by moving tool definitions and execution from your server to dedicated MCP servers. Instead of you authoring all the tools for a service like GitHub, an MCP Server for GitHub handles it.

The MCP Server wraps up functionality around a service and exposes it as a standardized set of tools. Your application connects to this MCP server instead of implementing everything from scratch.
