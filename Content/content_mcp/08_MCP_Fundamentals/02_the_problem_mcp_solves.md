## Content

Consider building a chat interface where users can ask Claude about their GitHub data. A user might ask "What open pull requests are there across all my repositories?" To handle this, Claude needs tools to access GitHub's API.

GitHub has massive functionality — repositories, pull requests, issues, projects, and more. Without MCP, you'd need to create an incredible number of tool schemas and functions to handle all of GitHub's features. This means writing, testing, and maintaining all that integration code yourself — a significant effort and ongoing maintenance burden.

MCP eliminates this by letting you connect to pre-built MCP servers that already handle the integration work.
