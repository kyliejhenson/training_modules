## Content

Prompts are triggered by user actions. Users decide when to run these predefined workflows through UI interactions like button clicks, menu selections, or slash commands. Prompts are ideal for implementing workflows that users can trigger on demand.

Use prompts when you want to create predefined workflows for users.
