## Content

Tools are controlled entirely by Claude. The AI model decides when to call these functions, and the results are used directly by Claude to accomplish tasks. Tools are perfect for giving Claude additional capabilities it can use autonomously.

When you ask Claude to "calculate the square root of 3 using JavaScript," it's Claude that decides to use a JavaScript execution tool to run the calculation.

Use tools when you want to give Claude new capabilities.
