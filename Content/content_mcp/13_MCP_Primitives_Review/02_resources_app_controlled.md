## Content

Resources are controlled by your application code. Your app decides when to fetch resource data and how to use it — typically for UI elements or to add context to conversations. Common uses:

- Fetching data to populate autocomplete options in the UI
- Retrieving content to augment prompts with additional context

Use resources when you want to get data into your app for UI or context purposes.
