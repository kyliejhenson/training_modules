## Content

A wide range of pre-built MCP servers are available for commonly used tools and services. The Anthropic MCP registry lists servers that are compatible with Claude Code, including integrations for services like GitHub, Sentry, Notion, Asana, and many more.

> **Important**: Use third-party MCP servers at your own risk. Anthropic has not verified the correctness or security of all servers. Make sure you trust the MCP servers you install. Be especially careful when using MCP servers that could fetch untrusted content, as these can expose you to prompt injection risk.

If you need a specific integration that doesn't exist yet, you can find hundreds more MCP servers on GitHub or build your own using the MCP SDK.
