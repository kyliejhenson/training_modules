## Content

Claude Code can connect to hundreds of external tools and data sources through the Model Context Protocol (MCP), an open source standard for AI-tool integrations. MCP servers give Claude Code access to your tools, databases, and APIs.

MCP acts as a bridge between Claude Code and the systems your team already uses — issue trackers, monitoring platforms, databases, design tools, communication apps, and more. Rather than switching between tools, you can ask Claude Code to interact with them directly.

MCP is an open source standard, meaning a growing ecosystem of pre-built integrations is available. You can also build your own MCP servers to connect Claude to proprietary internal systems.
